Instruções para execução:
	Abra o terminal dentro da pasta /dist
	Execute o comando "java -jar afn-to-er.jar <caminho do arquivo de entrada>"

Após a execução do comando, será criada duas pastas "dot" e "imagens".

É nelas que você irá encontrar os arquivos dot gerados e as imagens geradas pelo software.

Os estados iniciais vão estar coloridos de cinza e os finais vão ser um círculo duplo.

O JavaDoc está na pasta ./dist/javadoc, abra o arquivo index.html que está dentro dela.
